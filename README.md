# BoardBridge
A lightweight integration bridge connecting AutoDarts with Nakka N01 Web, enabling fully automated score synchronization between both platforms.

<b>KEY FEATURES</b>
- Real-time score synchronization from AutoDarts to Nakka N01 Web  
- Fully automatic score input without manual typing  
- Local setup designed for home use, clubs, pubs and live tournaments  
- Supports multiplayer X01 games  
- Automatic player switching  
- Checkout and game-shot detection support  
- No OCR or screen-reading required  
- No direct AutoDarts API integration required  
- Browser-based integration using Selenium automation  
- Lightweight event-driven architecture based on darts-caller output/logs


<b>MAIN ADVANTAGES</b>  

*Unlike OCR-based solutions, this approach does not rely on:*   

- screen recognition,  
- image processing,  
- coordinate tracking,  
- or fragile UI scraping.  

*This makes the integration:*    

- faster,  
- more stable,  
- more accurate,  
- and significantly easier to maintain.

*Intended Use Cases:*
  
- Home training setups  
- Local dart leagues  
- Pub tournaments  
- Streaming overlays and scoreboards  
- AutoDarts + Nakka hybrid tournament environments

<b>CURRENT STATUS</b>  

*Functional prototype with:*  

- automatic score synchronization,  
- checkout handling,  
- multiplayer support,  
- and live game integration already working successfully.  



<b>FLOW</b>  
1. Start AutoDarts match  
2. Launch darts-caller  
3. Run the bridge script  
4. Open Nakka N01 Web  
5. Scores are synchronized automatically in real time

<b>REQUIREMENTS</b>  
- AutoDarts  
- darts-caller  
- Python 3.8+  
- Selenium  
- Chrome / ChromeDriver  
- Nakka N01 Web

<b>INSTALLATION</b>  
pip install selenium  

Download matching ChromeDriver version:  
https://googlechromelabs.github.io/chrome-for-testing/  

Update the following variables in the script:  

ChromeDriver path  
darts-caller log path  
Nakka URL  

<b>ARCHITECTURE SECTION</b>  

The bridge listens to live match events generated by darts-caller, extracts scoring information in real time, and automatically forwards validated scores into Nakka N01 Web through browser automation.

<b>MOTIVATION</b>  

This project was created to combine the automatic scoring capabilities of AutoDarts with the statistics and tournament features of Nakka N01 Web.  

The goal was to create a lightweight and stable local tournament setup without relying on OCR, manual score input or unofficial API modifications. 

<b>CURRENNT LIMITATIONS</b>  
- Designed primarily for X01 game modes  
- Requires darts-caller output/log access  
- Browser automation currently targets Chrome  
- Initial Nakka game setup is manual

<b>Roadmap</b>
